# Bookshelf API (Express)


Project ready-to-run implementation meeting Dicoding Bookshelf API criteria.


## Cara pakai


1. Pastikan Node.js terinstall (v14+ direkomendasikan).
2. Jalankan:


```bash
npm install
npm start
```


3. Akses API di `http://localhost:9000` (server default port 9000). Jika ingin ganti port, set `PORT` environment variable.


## Endpoint
Lihat Postman collection yang kamu punya. Implementasi sudah mendukung semua route wajib + beberapa filter opsional: `?reading=1/0`, `?finished=1/0`, `?name=...`.
